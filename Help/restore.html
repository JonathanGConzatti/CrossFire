CrossFire Restore

Restore allows you to recover back ups of your Decks, Inventories, Combos, and Trades.

As each file is restored, its color will change from black to:

  • blue - successfully restored.
  • purple - file was unchanged since the back up.
  • green - the current version is newer than the back up.
  • red - appears to be a CrossFire file, but the type is unknown.

Card Warehouse Options

Report Format
Selects the default format for inventory reports.
  1. Full - Card number, name, rarity, quantity (with zeros in inventory reports).
  2. Verbose - Card number, name, rarity, quantity.
  3. Brief - Just the card numbers and quantities formatted by display method.
  4. HTML - Same as brief, but with HTML table tags.

Display Method
Specifies the way the list of card numbers and quantity will be displayed. Numbers in paranthesis is the quantity. The example lists are for 2 copies of card 1, 1 copy of card 2, and none of card 3.
  1. Brief: 1, 2
  2. Expanded: 1, 1, 2
  3. Normal: 1(2), 2
  4. Normal Verbose: 1(2), 2(1)
  5. Standard: 1(2), 2(1), 3(0)
  6. Inventory: 1-02, 2-01
  7. Full Inventory: 1-02, 2-01, 3-00

Auto Save
If checked, the inventory will automatically be saved after the specified number of minutes.

Card Sets
Selects which card sets will be available to Card Warehouse.

Inventory Directory
Changes the default directory for inventories.

Default Inventory
Selects the default inventory to use.

Reports Directory
Changes the default directory for generated reports.

Card Warehouse Change Wanted Quantity

Overview:

Change Wanted Quantity changes the maximum wanted quantity for all cards in the selected card set. Regular and chase cards are changed independently of each other.

Changing the Quantity:

There are two different methods to change the wanted quantity.

Enter an absolute value.
Example: 4 would change the wanted quantity for each card to 4.

Enter an amount to change by.
Example: +2 would add 2 to the current wanted quantity for each card.
Example: -1 would subtract 1 from the current wanted quantity for each card.
Example: +0 will leave the wanted quantity unchanged.
Card Warehouse will never allow the wanted quantity to be negative.

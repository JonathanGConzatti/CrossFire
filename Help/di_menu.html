DeckIt! Menu Bar

Deck

    New
    Creates a new deck. First checks if current deck needs to be saved. The new deck's size will be the default size set in the options.

    Open
    Loads a saved deck. First checks if current deck needs to be saved.

    Save
    Saves the current deck.

    Save As
    Saves the deck with a new filename.

    Information
    Allows for entering the author's name and email address and the title for the deck. This information is optional.

    Format
    Changes the deck format of the current deck. When changing to a smaller deck size, the card quantities and champion levels will be checked to make sure they are within the new limits.

    Multiple Cards
    Changes if multiple cards are allowed. When changing to not allowing multiple, the deck will be checked to make sure there are no multiples except for those that are always allowed. (3rd Edition War Party, etc.)

    Import
    Imports decks from other Spellfire utilities.
    1. Spellfire Database 2.x

    Export
    Exports the deck to other Spellfire utilities.
    1. Spellfire Database 2.x

    Print
    Unfortunately, Tcl/Tk does not provide support for printing, but a form of printing is available.

    All platforms: Can create a text file of the deck. Card text can optionally be printed; see options.

    On Windows and Macintosh: Creates a rich text document of the deck. Can be printed with WordPad or Word on Windows and Word on Macintosh.

    On Unix/Linux: Prints the deck to the default printer or to standard out.

    Exit
    Closes the deck editor. First checks if current deck needs to be saved.

Card

    These sub-menus may be "torn-off" by clicking on the line of dashes at the top to create a floating menu.

    Set
    Selects a card set. This can also be changed using the comma (,) or period (.) keys.

    Type
    Selects a card type.

Utilities

    Deck Differ
    Deck Differ compares two decks to find cards that are in both decks or cards that are only in one deck.

    Deck Status
    Deck Status shows current quantity and limit of each card type.

    Ultra Searcher
    Ultra Searcher allows for very precise card searches.

    Configure
    Change DeckIt! options.

Help

    Help
    Starts the hypertext help system.

    About
    Brief tribute to the author of DeckIt!

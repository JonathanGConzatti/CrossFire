Fan Set Editor Menu Bar

Card

    Save
    Saves the current card.
    Auto Save
    If checked, the card will automatically be saved without prompting you.
    Move
    Moves the current card to another location. The current card will then be blank and the new location will be overwritten with the card information.
    Clear
    Clears all the fields of the card.
    Close
    Closes the fan set editor. First checks if current card needs to be saved.

Utilities

    Configure
    Change the Fan Set Editor's options.

Help

    Help
    Starts the hypertext help system.
    About
    Brief tribute to the author of the Fan Set Editor.

Launcher Options

Graphical Main Menu
Changes the main CrossFire window to use icons.

Background Image
Toggle for display of the background image.

Mouse-Over Help
Toggle for display of the name of the CrossFire program when the mouse is moved over the icon.

Icon Set
Selects the set of icons to display in grahical mode. Visit the CrossFire Resources page to download new sets.

Show Icons For
The Launcher can be customized to only display the icons for the parts of CrossFire you use the most. Simply select or deselect to toggle the display of the program's icon.

Ultra Searcher

Overview:

The Ultra Searcher was created to allow for very precise card searches. For example: you wanted to find all undead monsters or all Greyhawk realms with the word "immune" in the card text. These kinds of searches can now be done!

Searching:

Each part of the Ultra Searcher dialog has a check box. Select that check box to limit the search to what is selected in the lists or entered in the fields. The items in a list are OR (not AND) searches! (ie: selecting Flyer, Awnshegh, and Swimmer will match any of those.) Caution: <Clear> clears the entire form. You can add cards from the results box to a deck by draging and dropping the card onto the deck list.

Fields:

Card Title and Text
Enter plain text or a regular expression. This text is NOT case sensitive.

Levels
The level field is entered as a list of levels to search for. It also allows for ranges of levels such as 2-5, or <5 or >8. These may be combined: <4, 7, 8, 10-12. This will find all levels less than 4 or equal to 7, 8, 10, 11, or 12.

Last Digit
Select the desired last digit of the card number. Ranges can be specified the same way as "Levels" above.

Attributes, Card Types and Worlds
Select the desired criteria from the list.

Example:

  1. All Undead Monsters:
    1. Push <Clear>
    2. Click Attributes
    3. Select Undead
    4. Click Card Types
    5. Select Monster
    6. Click <Search>
  2. All Greyhawk realms with the word "immune" in the card text:
    1. Push <Clear>
    2. Click Card Text
    3. Type 'immune' into the entry field. (This is NOT case sensitive.)
    4. Click Card Types
    5. Select Realm
    6. Click Card Worlds
    7. Select Greyhawk
    8. Click <Search>
    This was to (hopefully) find the realms that were immune to something or inferred immunity to champions. A better text to type might be "immun".

DeckIt! Options

Deck Format
Changes the format of deck. The standard 55, 75, and 110 card decks are supported. Additionally, an 'Unlimited' deck is supported. This deck type is mainly used to pick many cards that *may* be used in a deck and then editting down to a standard deck type. Use caution when going to a smaller size deck; the editor will not allow you to do so if any card quantity will be violated. The default is a 55 card deck.

Display By Card ...
Selects the method of deck display and printing. The choices are:
  1. Card Type
  2. World
  3. Card Set
  4. Rarity
Default is type.

Group Champions Together
Selects if champions should be grouped under one heading. This only works for the display by type mode. Default is to not print the card text.

Print Card Text
Selects if the card text should be printed with the card names. Default is to not print the card text.

Auto Save
If checked, decks will automatically be saved after the specified number of minutes.

All Card Sets
Selects which card sets will be used when "All Card Sets" selected.

Deck Directory
Selects the default directory DeckIt! will use to store decks.

Card Warehouse Menu Bar

File

    New
    Creates a new inventory. First checks if current inventory needs to be saved.

    Open
    Loads a saved inventory. First checks if current inventory needs to be saved.

    Save
    Saves the current inventory.

    Save As
    Saves the inventory with a new filename.

    Deck: Remove From Inventory.
    Removes all the cards in a deck from the current inventory. This command changes the "in inventory" status of the deck to "no" and saves the adjusted inventory.

    Deck: Add To Inventory.
    Adds all the cards in a deck to the current inventory. This command changes the "in inventory" status of the deck to "yes" and saves the adjusted inventory.

    Import
    Imports a Spellfire Database 2.x inventory. The importing is performed from a backup (spellfire.bck) file made by Spellfire Database. Do not try to import a CSV created by Spellfire Database 2.0-2.3...there is a card text error that causes problems. You may also import from Spellfire Collector. Export the inventory to Spellfire Database which creates a CSV file (spellfire.csv), then import from that file.

    Export
    Exports a Spellfire Database 2.x inventory. Exporting creates a backup file for Spellfire Database to restore from.

    Exit
    Closes the inventory editor. First checks if current inventory needs to be saved.

Card

    Set
    Selects a card set. This can also be changed using the comma (,) or period (.) keys.

    View
    Views the selected card.

Report

    Inventory
    Generates an inventory report of the current card set in the selected format and display method.

    Wanted Cards
    Generates a report of the wanted cards of the current card set in the selected format and display method.

    Extra Cards
    Generates a report of the extra cards of the current card set in the selected format and display method.

    Format
    Changes the format of the report. See options.

    Display Method
    Changes the way card number and quantity are displayed for Brief and HTML formats. See options.

Utilities

    Card Counter
    Card Counter reports the number of cards in a set.

    Change Wanted Quantity
    Change Wanted Quantity allows for setting or adjusting the maximum wanted quantity of all the cards in a set.

    Change Multiple Card Quantities
    Change Multiple Card Quantities allows for changing many cards with one command.

    Configure
    Change Card Warehouse options.

Help

    Help
    Starts the hypertext help system.

    About
    Brief tribute to the author of the Card Warehouse.

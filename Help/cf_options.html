CrossFire Options

Name
Enter your name.

Email Address
Enter your email address.

Booster Order
Selects the way boosters will be listed.

The choices are:

  1. Alphabetical: alphabetical order by set name.
  2. Chronological: listed in the order they were released.
Default is Chronological.

Memory Usage
Selects the amount of memory to use for caching the database.

The choices are:

  1. Small: does not cache at all.
  2. Medium: caches each individual card as it is used.
  3. Large: caches the whole set a card is from.
Default is Small.

Use Big Windows
Adds a card set selection window to some of the CrossFire applications.

Fan Set Editor Introduction

The Fan Set Editor is used to edit the cards in a fan created set.

There are four main areas of the Fan Set Editor:

  1. The Menu Bar

  2. Author Name and Email

    This shows the author's name and email address.

  3. Card Selection List

    The card selection list contains all the cards in the set. While you can edit any of the fan cards, obviously the intent of this editor is for editing the set you created!

  4. The Card Editor

    This displays the selected card for editing.
    Some general notes to follow:
    • Be logical. ie: Don't create an Avatar Artifact, etc.
    • Enter bonuses exactly as they should appear. ie: +3/+6, +?, -2, 5/7.
    • Include "No limit per deck." in the card text for unlimited copies of the card.
    • Include "Limit x per deck." to allow x number of copies. ex: Limit 3 per deck.
